# Solar Billing - Cultivve Gestão

Sistema de cobrança de energia solar com leitura automática de contas Energisa.

## Funcionalidades

- Upload de PDF da conta Energisa com extração automática dos dados
- Cálculo do consumo instantâneo, total e preço efetivo por kWh
- Geração de mensagem WhatsApp para cobrança
- Histórico mensal de cobranças
- PWA — funciona como app no iPhone

## Como usar no iPhone

1. Acesse o link do GitHub Pages no Safari ou Chrome
2. Toque no ícone de compartilhar → **Adicionar à Tela Início**
3. Use como um app nativo

## Tecnologias

- HTML/CSS/JS puro (arquivo único)
- PDF.js para leitura local de PDFs
- LocalStorage para histórico
- PWA com manifest.json
